# Cloud HIS

#### 介绍
本设计的主要任务是基于云计算的基础设施和微服务的分布式技术，完成医院信息系统中各个服务模块的构建、部署、组装和管理。该云HIS系统，主要包括医院门诊住院业务的挂号收费、医护工作站、医技、药房等核心模块的设计与实现，保证各个模块流程的衔接，同时系统架构实现业务框架与技术设施的完全解耦合，可以达到低成本高效率地对接其他平台的业务模块和边缘设施。
